Which tools to use for Data Science
===================================

:Priority: MEDIUM


Rationale
---------

**As a** `developer`, **I** need to define what tools to use to do data
analysis, **so that** we can identify all possible patterns and start
developing all implied tools.


Description
-----------

There are a more elaborated list to evaluate with support for data analysis:

- `pandas`_: data analysis and manipulation tool.
- `SciPy`_: mathematics, science, and engineering (optimization, integration,
  interpolation, eigenvalue problems, algebraic equations, differential
  equations, statistics, etc).
- `scikit-learn`_: a module for machine learning built on top of SciPy_.

.. _pandas: https://pandas.pydata.org/
.. _scipy: https://scipy.org/
.. _scikit-learn: https://scikit-learn.org
